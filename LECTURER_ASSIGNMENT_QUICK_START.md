# Lecturer Assignment Management - Quick Start Guide

## Accessing the Feature

1. **Login** as a lecturer
2. **Navigate** to "Assignments" from the sidebar menu
3. You'll see the assignment management dashboard

## Creating an Assignment

### Step 1: Click "Create Assignment"
- Blue button in the top-right of the header card
- Form will appear with two tabs

### Step 2: Fill Assignment Details (Tab 1)

#### Basic Information:
1. **Assignment Title**: Enter a descriptive title (e.g., "Data Structures Quiz 1")
2. **Description**: Explain the assignment purpose and scope

#### Academic Information:
3. **Subject**: Select from YOUR teaching subjects only
4. **Modules**: Select one or more modules (loaded based on subject)
5. **Due Date**: Pick a date using the date picker

#### Assignment Configuration:
6. **Assignment Level**: Choose Easy, Medium, or Hard
7. **Assignment Type**: Choose MCQ, Short Answer, or Essay
8. **Number of Questions**: Enter count (1-100)
9. **Maximum Marks**: Will auto-calculate after question generation
10. **Submission Type**: Online, File Upload, or Both
11. **Time Limit**: Optional (in minutes)

#### Question Generation Source:
12. Choose one:
    - **Generate from module names**: AI uses module titles
    - **Provide custom content**: Paste your own content

#### Additional Settings:
13. **Instructions**: Optional special instructions for students
14. **Allow Late Submission**: Toggle on/off
15. **Late Penalty**: Percentage if late submissions allowed

### Step 3: Generate Questions
1. Click **"Generate & Preview Questions"** button
2. Wait for AI to generate questions (shows spinner)
3. Automatically switches to Tab 2

### Step 4: Review Questions (Tab 2)
1. **Expand** each question panel to review:
   - Question text
   - Options (for MCQ)
   - Correct answers
   - Marks per question
2. **Total marks** displayed at top
3. Questions are color-coded by type

### Step 5: Save Assignment
1. Click **"Save Assignment"** button in Tab 2
2. Success notification appears
3. Assignment appears in table
4. Form automatically closes

## Editing an Assignment

1. **Find** the assignment in the table
2. Click **Edit icon** (pencil) in Actions column
3. Form opens with all fields pre-filled
4. **Modify** any fields as needed
5. Click **"Generate & Preview Questions"** if you changed modules/settings
6. Click **"Update Assignment"** to save changes

## Deleting an Assignment

1. **Find** the assignment in the table
2. Click **Delete icon** (trash) in Actions column
3. **Confirm** deletion in dialog
4. Assignment removed from table

## Toggling Assignment Status

1. **Find** the assignment in the table
2. Use the **toggle switch** in Status column
3. Active (green) = Students can see it
4. Inactive (gray) = Hidden from students

## Filtering Assignments

Use the Filters card to narrow down the list:

1. **Subject**: Show assignments for specific subject
2. **Assignment Type**: Filter by MCQ, Short Answer, or Essay
3. **Level**: Filter by Easy, Medium, or Hard
4. **Status**: Filter by Active or Inactive
5. Click **"Clear"** to reset all filters

## Understanding the Table

### Columns:
- **Title**: Assignment name and description snippet
- **Subject**: Subject name
- **Batch**: Student batch
- **Semester**: Academic semester
- **Type**: Badge showing question type
- **Level**: Badge showing difficulty
- **Due Date**: Submission deadline
- **Questions**: Count and total marks
- **Status**: Active/Inactive toggle
- **Actions**: Edit and Delete buttons

### Color Coding:
- **MCQ**: Blue badge
- **Short Answer**: Green badge
- **Essay**: Purple badge
- **Easy**: Green badge
- **Medium**: Yellow badge
- **Hard**: Red badge

## Pagination

- **Items per page**: Choose 5, 10, 25, or 50
- **Navigation**: Use arrows to move between pages
- **Total count**: Displayed in table header

## Tips & Best Practices

### Creating Effective Assignments:

1. **Clear Titles**: Use descriptive names with subject and topic
   - Good: "DSA201 - Binary Trees Quiz"
   - Bad: "Quiz 1"

2. **Detailed Descriptions**: Explain what the assignment covers
   - Topics included
   - Learning objectives
   - What to prepare

3. **Module Selection**: 
   - Select relevant modules only
   - More modules = broader question pool
   - 2-3 modules recommended for focused assignments

4. **Question Count**:
   - MCQ: 10-20 questions for quizzes
   - Short Answer: 5-10 questions
   - Essay: 2-5 questions

5. **Difficulty Level**:
   - Start with "Easy" for introductory topics
   - Use "Medium" for regular assessments
   - Reserve "Hard" for advanced students

6. **Time Limits**:
   - MCQ: 1-2 minutes per question
   - Short Answer: 5-10 minutes per question
   - Essay: 15-30 minutes per question

7. **Late Submission**:
   - Enable for homework assignments
   - Consider 10-20% penalty
   - Disable for time-sensitive quizzes

### Question Generation:

1. **Custom Content**:
   - Paste lecture notes or textbook excerpts
   - Include code examples for programming subjects
   - Provide diagrams descriptions if relevant

2. **Review Carefully**:
   - Always preview questions before saving
   - Check for clarity and correctness
   - Verify answer keys for MCQs
   - Ensure mark distribution is fair

3. **Regeneration**:
   - If questions aren't satisfactory, click "Generate" again
   - Try different module combinations
   - Adjust difficulty level

### Managing Assignments:

1. **Timing**:
   - Create assignments at least 3 days before due date
   - Set realistic due dates
   - Consider student workload

2. **Status Management**:
   - Keep active only during assignment period
   - Deactivate after deadline if no late submissions
   - Archive old assignments by deactivating

3. **Organization**:
   - Use consistent naming conventions
   - Group by module/topic in titles
   - Add semester/year in description

## Common Issues & Solutions

### Issue: No modules appear when selecting subject
**Solution**: The subject may not have any modules yet. Contact admin or create modules first.

### Issue: Question generation fails
**Solution**: 
- Check if modules have content
- Try selecting different modules
- Use custom content option instead

### Issue: Can't see some subjects
**Solution**: You can only see subjects assigned to you. Contact admin if a subject is missing.

### Issue: Assignment doesn't appear in table
**Solution**:
- Check filter settings
- Verify assignment was saved (look for success message)
- Refresh the page

### Issue: Students can't see assignment
**Solution**: Check if status toggle is set to Active (green)
